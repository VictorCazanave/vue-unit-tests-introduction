# Vue Unit Tests Introduction

A random project to present how to write unit tests for [Vue.js](https://vuejs.org/) with [Jest](https://jestjs.io/).

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Run your tests

```
npm run test
```

### Lints and fixes files

```
npm run lint
```
